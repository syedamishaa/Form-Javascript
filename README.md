# 📋 Student Entry Form 

A Student Entry Form project built using HTML, CSS, and JavaScript with DOM Manipulation.

## 📌 Project Overview

This project allows users to enter student information and display it dynamically in a table without refreshing the page.

The project was developed to practice JavaScript DOM Manipulation.

## ✨ Features

* Add Student Information
* Display Student Records
* Delete Student Records
* Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

### CSS

* Responsive Design

### JavaScript

* DOM Manipulation
* Event Handling
* Dynamic Content Creation
* Form Handling

## 🚀 Learning Outcome

This project helped me understand how data can be collected from forms, manipulated using JavaScript, and displayed dynamically on a webpage.

It also provided hands-on experience with Create, Read, Update, and Delete (CRUD) operations.
